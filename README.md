e-Ballot — Online Voting System
A secure, web-based online voting platform that allows citizens to vote for their preferred candidates from anywhere, reducing political conflict and making the democratic process more accessible.

📌 Project Overview
e-Ballot is an online voting system developed to eliminate the physical barriers of traditional voting. Citizens living abroad or in different cities can vote safely from home using their National ID (NID). The system prevents duplicate votes by linking each account to a unique NID card.
